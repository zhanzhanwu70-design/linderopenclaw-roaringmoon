### upstream/main 新增（dev/rust 領先 main 的14個commits）

**重大新功能：**

#### 1. HTTP/SSE Server (`rust/crates/server/`)
- 基於 axum 的 HTTP/SSE 服務端
- Session 管理：create/list/get/update sessions
- SSE event streaming：snapshot + message 事件
- 使用 `tokio::sync::broadcast` channel 實現事件廣播
- 使用 `async_stream` 產生 async stream
- 端點：
  - `POST /sessions` - 建立 session
  - `GET /sessions` - 列出所有 sessions
  - `GET /sessions/{id}` - 取得 session 詳情
  - `POST /sessions/{id}/message` - 發送訊息（返回 NO_CONTENT）
  - `GET /sessions/{id}/events` - SSE event stream
- `RuntimeSession` 用於存儲對話狀態

#### 2. Provider 抽象層 (`rust/crates/api/src/providers/`)
- 新 trait `Provider`：`send_message()` + `stream_message()`
- `ProviderKind`: ClawApi, Xai, OpenAi
- `MODEL_REGISTRY`: 靜態映射 model name → provider/auth/base_url
- 支援 API key + Bearer token 雙重認證 (`AuthSource`)
- 實作：`claw_provider.rs`（Anthropic API）、`openai_compat.rs`（OpenAI compatible）
- OAuth token 管理：`OAuthTokenSet`
- 背off retry 邏輯（initial 200ms, max 2s, max 2 retries）

#### 3. Python Porting Workspace (`src/`)
- 完整 Python 重新實現 workspace
- `src/main.py`: CLI entrypoint，含 subcommands
- `src/parity_audit.py`: 對比 Python workspace 與存檔的 TypeScript 原始碼
- `src/port_manifest.py`: 追蹤 port 進度
- `src/query_engine.py`: 查詢 engine
- `src/runtime.py`: Python runtime 模擬
- `src/session_store.py`: Session 持久化
- `src/tool_pool.py`: Tool pool 管理
- `src/remote_runtime.py`: 遠程 runtime 模擬（remote/ssh/teleport/direct-connect modes）
- `reference_data/`: TypeScript 原始碼 snapshot 對照數據
  - `commands_snapshot.json`: 1037 個命令条目
  - `tools_snapshot.json`: 922 個工具条目
